Patroni is a template to create your own high-availability solution, relying on PostgreSQL's streaming replication and hot standby capabilities, using Python and - for maximum accessibility - a distributed configuration store like etcd or Kubernetes.

!!! Note
    Looking for Patroni documentation? Head over to [https://patroni.readthedocs.io](https://patroni.readthedocs.io/en/latest/)
