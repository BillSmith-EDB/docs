---
title: "Version 7.4.0"
---


New features, enhancements, bug fixes, and other changes in Replication Server 7.4.0 include the following:

| Type    | Description |
| ------- |------------ |
| Enhancement  | EDB Replication Server is now certified to work with the new major release of PostgreSQL version 15. PostgreSQL 15 is now supported both as a source Publication and/or target Subscription database in trigger and WAL based replication cluster configurations.  |
| Bug fix      | Fixed an issue where a redundant duplicate copy of `commons-lang3-3.12.0.jar` is placed as part of the EDB Replication Server installation on Windows OS. |
