The in-app help provides comprehensive details about PEM web interface including items in the drop down menu bars, charts and graphs on the dashboards, and creating database objects using dialog boxes.

- To access the landing page of the help, select **Help > Online help** menu. 
- To access the context sensitive help for a dialog, select the **?** button.
