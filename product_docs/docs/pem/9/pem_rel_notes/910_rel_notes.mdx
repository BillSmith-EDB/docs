New features, enhancements, bug fixes, and other changes in PEM 9.1.0 include:

| Type        | Description                                                                                                                                                  | ID                 |
| ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------ |
| Enhancement | Support for EDB Advanced Server 15 and EDB Postgres Distributed 5.                                                                                           | PEM-4640           |
| Enhancement | Added the ability to copy probes and alerts to all servers in a group without having to select them individually.                                            | PEM-4613, PEM-4681 |
| Bug Fix     | Fixed an issue whereby alert notifications were sent for Low or Medium alerts when the user had selected not to receive them. [Support Ticket #87664]        | PEM-4650           |
| Bug Fix     | Fixed an issue whereby webhooks would not be called if there was an inactive endpoint in the spool. [Support Ticket #87365]                                  | PEM-4664           |
