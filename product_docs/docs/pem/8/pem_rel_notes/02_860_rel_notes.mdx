title: "Version 8.6.0"
---

New features, enhancements, bug fixes, and other changes in PEM 8.6.0 include:

| Type         | Description                                                                                                                                                                                                                                                          | ID        |
| ------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------| --------- |
| Enhancement  | The team support for the database servers, agents, and tools is improved. When no team is specified for the particular object, you can control the visibility of the database servers, agents, and tools by configuring the `show_objects_with_no_team` parameter.   | PEM-2843  |
| Enhancement  | Differentiate the Postgres Extended servers in the core usage report.                                                                                                                                                                                                | PEM-4212  |
| Enhancement  | Added support for monitoring the PostgreSQL 15, EDB Postgres Extended Server 15 & EDB Postgres Extended Server 15.                                                                                                                                                   | PEM-4544  |
| Bugfix       | Fixed an issue parsing the BARMAN `diagnose` API.                                                                                                                                                                                                                    | PEM-4517  |
| Bugfix       | Fixed an issue of purging operation for the probes data of the `extension` type was resulting in to an error.                                                                                                                                                        | PEM-4576  |
