---
title: "Changing the default port"
---

By default, the web service is listening on the port 8443. You can change the default port as per your requirement.

For Debian 10, Ubuntu 18 or 20:

- In the file `/etc/apache2/sites-available/edb-ssl-pem.conf`, change the following parameters with your required port number:

    ```text
    Listen 8443

    <VirtualHost _default_:8443>

    ServerName HTTPD-EDBPEM-SERVER-v8:8443
    ```

 
- If the file `/etc/apache2/sites-available/edb-gnutls-pem.conf` exists then change the following parameters with your required port number:

    ```text
    Listen 8443

    <VirtualHost _default_:8443>

    ServerName localhost:8443
    ```


- In the file `/etc/apache2/sites-available/000-default.conf`, change the port number(8443) to your required port number:

    ```text
    Redirect "https://%{HTTP_HOST}:8443%{REQUEST_URI}"
    ```
