<div id="installing_mtk" class="registered_link"></div>

Before installing Migration Toolkit, you must install Java (version 1.8.0 or later).

You can install Migration Toolkit on:

- [Linux x86-64 (amd64)](install_on_linux/x86_amd64) and [IBM Power (ppc64le)](install_on_linux/ibm_power_ppc64le)
- [Windows x86-64](install_on_windows)
- [Mac OS X](install_on_mac)


<div id="installing_drivers" class="registered_link"></div>

## Installing EDB JDBC drivers

If your source or target will be EDB Postgres Advanced Server, you must download and install a JDBC driver provided by EDB before invoking Migration Toolkit.  To download the latest driver, see [EDB Connectors](https://enterprisedb.com/software-downloads-postgres#edb-connectors) on the Downloads page.

After downloading the EDB driver, move the driver file into the `<mtk_install_dir>/lib` directory.

## Installing third-parthy JDBC drivers

If your source will be MySQL, SQL Server, or Oracle, you must download and install a freely available source-specific driver before invoking Migration Toolkit. For a link to a vendor download site, see [Third Party JDBC Drivers](https://www.enterprisedb.com/software-downloads-postgres#third-party-jdbc-drivers) on the Downloads page.

After downloading the source-specific driver, move the driver file into the `<mtk_install_dir>/lib` directory.
