---
title: Postgres extensions
---

BigAnimal supports many Postgres extensions. See [Supported Postgres extensions](/pg_extensions/latest/) for the complete list.

## Installing and removing extensions

Use the CREATE EXTENSION command to install extensions. See [CREATE EXTENSION](https://www.postgresql.org/docs/current/sql-createextension.html) for more information.

Use the DROP EXTENSION command to remove extensions. See [DROP EXTENSION](https://www.postgresql.org/docs/current/sql-dropextension.html) for more information.

## Extensions requiring superuser privileges

[Supported Postgres extensions](/pg_extensions/latest/) indicates which extensions require superuser privileges to install. In addition, certain extensions require superuser privileges in order to use the extension:

| Extensions requiring superuser to use |                             Notes                             |
| ------------------------------------- | ------------------------------------------------------------- |
| amcheck                               | Can be granted to others                                      |
| pageinspect                           |                                                               |
| pg_buffercache                        | superuser or pg_monitor role                                  |
| pg_freespacemap                       | superuser or pg_stat_scan_tables role                      |
| pgrowlocks                            | superuser or pg_stat_scan_tables role                      |
| pgstattuple                           | superuser or pg_stat_scan_tables role                      |
| pg_visibility                         | superuser or pg_stat_scan_tables role                      |
