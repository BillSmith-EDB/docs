---
title: Postgres extensions
---

BigAnimal supports many Postgres extensions. See [Supported Postgres extensions](/pg_extensions/latest/) for the complete list.

## Installing extensions

You use the CREATE EXTENSION command to install extensions. See [CREATE EXTENSION](https://www.postgresql.org/docs/current/sql-createextension.html) for more information.

## Extensions requiring superuser access

Certain extensions require superuser permissions in order to install and access the extension:

| Extensions requiring superuser access |                             Notes                             |
| ------------------------------------- | ------------------------------------------------------------- |
| adminpack                             | Can be granted to others                                      |
| amcheck                               | Can be granted to others                                      |
| file_fdw                              |                                                               |
| pageinspect                           |                                                               |
| pgagent                               | Requires running a daemon local to your container             |
| pg_buffercache                        | superuser or pg_monitor role                                  |
| pg_freespacemap                       | superuser or or pg_stat_scan_tables role                      |
| pgrowlocks                            | superuser or or pg_stat_scan_tables role                      |
| pgstattuple                           | superuser or or pg_stat_scan_tables role                      |
| pg_visibility                         | superuser or or pg_stat_scan_tables role                      |
| spi                                   | SPI includes refint, autinc, moddatetime, and insert_username |
