---
title: "Limitations"
---

Columns of collection types are not supported.

This example shows the error displayed when we try to create the column of collection type:

```sql
--Use varray in composite type and then use it to create table 
create type subarray is varray(10)of number;
create type footype as (x int, y subarray);
create or replace package subpack is
  subtype subfoo is footype;
end;
--Create column of collection type
create table subtest_1(p footype);
__OUTPUT__
ERROR:  column "p" has collection type footype, columns of collection types are not supported.
```
